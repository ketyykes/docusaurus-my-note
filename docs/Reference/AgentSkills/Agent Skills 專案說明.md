# Agent Skills 專案說明

## 前言

[Agent Skills](https://agentskills.io) 是一種簡單、開放的格式，用於賦予 AI 代理新的能力與專業知識。

Skills 是由指令、腳本和資源組成的資料夾，讓代理能夠自動發現並用來更好地執行特定任務。編寫一次，到處使用。

## 快速入門

- [官方文件](https://agentskills.io) — 指南與教學
- [格式規格](https://agentskills.io/specification) — 格式詳細說明
- [範例 Skills](https://github.com/anthropics/skills) — 查看有哪些可能性

此儲存庫包含規格文件、官方文件與參考 SDK。範例 Skills 清單請參閱[此處](https://github.com/anthropics/skills)。

## 關於本專案

Agent Skills 是一個由 [Anthropic](https://anthropic.com) 維護的開放格式，歡迎社群貢獻。

Agent Skills 是由 Anthropic 發起的開源專案，提供一種開放格式讓 AI 代理學習新技能，採用 Apache 2.0 與 CC-BY-4.0 雙重授權。
